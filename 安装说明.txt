书签管理器 - 安装说明
=====================

一、直接安装（推荐）
----------------

1. 在 Chrome 浏览器地址栏输入：chrome://extensions/
2. 打开右上角的"开发者模式"开关
3. 点击左上角的"加载已解压的扩展程序"按钮
4. 选择本项目的 dist 文件夹
5. 安装完成！

二、使用方法
-----------

1. 点击浏览器右上角的书签管理器图标
2. 点击"整理书签"按钮，自动分类整理书签
3. 使用其他功能：
   - 重复检测：查找并删除重复书签
   - 链接检查：检测失效的链接
   - 导入导出：备份或恢复书签
   - 云端同步：同步书签到云端

三、功能说明
-----------

✓ 智能分类：自动将书签分类到相应文件夹（搜索引擎、邮箱、社交媒体等）
✓ 重复检测：识别和删除重复的书签
✓ 链接检查：检测书签是否失效
✓ 云端同步：支持书签云端备份和同步
✓ 导入导出：支持 HTML 和 JSON 格式导入导出

四、常见问题
-----------

Q: 整理书签时只整理了根目录的书签？
A: 是的，插件只会整理根目录下的书签，不会重新整理已经分类好的书签。

Q: Gmail 没有被分类到邮箱？
A: 请确保 Gmail 书签在根目录，然后点击"整理书签"按钮。

Q: 游戏分类误判了其他网站？
A: 分类规则已优化，如果仍有误判，可以手动移动书签到正确文件夹。
